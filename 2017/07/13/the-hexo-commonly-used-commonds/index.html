博客更新常用命令

前言

之前自己搭建了一个博客，好久没有更新了，有些命令也快忘记了，还是需要写下防止忘记。

新建博文

hexo new ‘the_name_of_the_article’

生成静态页面文档

hexo g

启动本地服务预览

hexo server

部署网站到github

hexo d
