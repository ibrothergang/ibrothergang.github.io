目前在用的第三方服务

发表于 2017-02-19

一家创业公司，没有大公司的资源和财力，可以所有的东西都自己搞。所以除了核心技术和服务，最快的方式就是依靠第三方的服务快速的搭建自己的产品。

统计

决定一个产品好不好用，很多时候是靠数据说话的。什么日活，月活，新增，留存等等，没有统计哪里来的数据。所以数据统计这块是相当重要的。目前我所接触到的主要是下面这几家。

友盟

应该算国内的老牌了吧，最大的优势统计应该是统计的事件能基本做到实时上传生成报表。这对于开发来说十分方便。总不能让我添加完事件后过两天才能看到数据吧。

ThinkingData

这个没有用过，暂时不评论。

Flurry

这个是海外的数据统计平台，最大的特点应该是支持漏斗分析，这个对运营来说特别有用。转化率什么的再也不用拿数据一步步计算了，直接设置完漏斗，数据就出来了。数据统计上也比较完整。

GA(Google Analysis)

GA就不说了，现在也只能叫海外的统计平台了。用户属性方面比较全，毕竟这么大的数据量，随便分析下就能获取用户的画像信息。但是事件统计这块真心不怎么样，数据的真实性也表示怀疑。曾经发生过添加了一个统计事件导致整个新增数量飙升的情况。去掉后数据又回落了，从此对这块数据就持怀疑态度了。

推送

现在的App，没有推送功能都不好意思说自己是个完整的App。如果没有关闭推送功能，往往是推送消息满天飞。在Android上，这种情况尤为突出。因为Android手机不像iOS，iOS的推送都是需要走苹果的通道，所以用户也可以在设置中主动关闭，但是Android一般情况下不可以（这部分权限ROM被重新定制过的手机除外）。这其实是一个很好的现象，直接的结果就是导致体验比较差，很有可能用户闲烦就把你卸载了。
跑偏了，合理的推送对用户来说还是十分必要的。

GCM(Google Cloud Message) & FCM(Firebase Cloud Message)

FCM是GCM的升级版本，Google在2016年的时候大力推了一把Firebase，可惜在国内没多少人买它的账。FCM和GCM一个共同的特点是后台搭建比较麻烦，而且到现在为止我们也不知道怎么样去获取推送的送达数。没有数据的统计就无法衡量效果，所以目前为止我们还是在用国内的推送服务商。

极光推送（JPush）

这个算老牌了吧，推送一直算比较稳定吧，但是到达率也不算高，不知道是不是没有付钱，使用的是免费版本的原因。

小米推送（MiPush）

在小米的设备上到达率特别高，因为小米推送在小米设备上是系统级的服务。类似于Android原生手机上的GCM和FCM.

个推

这个算后起之秀吧，国内到达率应该挺高的。杭州这边的公司普遍到达率有90%以上。我们的用户大部分在海外，所有到达率也没有那么高。

App中使用的场景不多，所以也没有深入研究，满足基本的定位和获取城市信息即可。

私信从本质上讲就是一个个的长连接。和push的原理很类似。环信和融云其实也差不多，最终就看价格和服务了。

谈谈过去的一年

发表于 2017-01-24 | 分类于 随笔

谈谈过去的一年

记得小时候总是盼着自己快快长大，能够做一些长大了能做的事情。随着慢慢的长大，发现小时候想做的事情做到了，但是又有更多的事情想去做。
随着年龄的增长，觉得时间越来越短。有人说，那是因为你真正长大了，但是我觉得，可能是自己的欲望变强了，想要的东西变多了。不在是以前那个吃个棒棒糖就很开心，守着一个沙土堆就能玩半天的懵懂小孩了。
到这个时间点，多多少少还是需要停下脚步，看看自己过去的一年，都走了什么路。

纯技术转为半管理

以前的自己，专注的都是技术。后来团队人员越来越多，慢慢的管理方面的事情就变得越来越多了，技术上的专注就变少了。

公司搬家

2016年4月16日，这是一个值得纪念的日子。这一天，公司经历了又一次搬家。那一刻的心情的确是比较复杂的。
公司办公环境从一个复式的小作坊开始，到后来几百平的写字楼，再到现在的整个一层楼；队伍从一只10个人不到的开始，慢慢的成长起来，到现在的100来号人。这是一个积累的过程，虽然不能以快速来形容，但是始终是一个逐渐成长的过程。

二宝出生

2016年4月27日，这又是一个值得纪念的日子。这一天，我们第二个小宝宝诞生了。她一一直在我们的期待中。之前对于这个小生命其实是充满了幻想，想象着会和哥哥一样吗？也会很调皮吗？以后会和哥哥一起玩的开心吗？
当小生命诞生的时候，一切的担忧都没有必要了，唯一希望的是小家伙能够平平安安，健健康康的成长起来。

大宝上幼儿园

2016年9月6日，这个日子对于大宝来说，具有特殊的意义，因为他上幼儿园了。虽然之前也上过幼儿园，但说实话就是村子里把小朋友聚在一起玩耍而已，这次是真真切切的要上一把幼儿园了。记得去的当天，这小子哭的伤心啊。不够其他小朋友也不见的有多好，幼儿园那天哭声一直断断续续的回荡。
还有一个就是臭小子终于和爸爸妈妈在一起了。最早的时候奶奶身体不好，久放在外婆家。后来外婆颈椎出问题了，就又去奶奶家了。辛亏这小子在这两个地方不怕生，住哪里都还习惯。而且也不是特别粘爸爸妈妈，这一点不像我，每次我妈回来看我，走的时候我都是会大哭一场，感觉世界上已经没有爱了。这小子不会！

团队扩充

公司现在已经超过百人了，我们Android团队也有新人加入进来。新人的加入，一方面促进了项目更好更快的发展，另一方面也给团队带来了新鲜血液。随着新人的加入，团队更能不断焕发新的活力。
